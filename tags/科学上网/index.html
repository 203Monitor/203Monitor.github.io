SSH Socks5

这是个很犀利的招式，windows怎么使用网上太多了，我就不说了，mac的用法更是简单，不过这一招并不在于怎么使用，前期攒起来才是最重要的。
原料：Mac电脑一台，openshift／koding帐号一个

  • ssh -qTfnN -D port xxx@xxx.xx.x
  • 打开浏览器，使用socks5代理，地址127.0.0.1:port

注：关闭步骤

  • lsof -i tcp:端口号
  • sudo kill -9 PID

此招式缺点：一小时断一次，不过别太沉迷于墙外的世界可以规避这个风险
